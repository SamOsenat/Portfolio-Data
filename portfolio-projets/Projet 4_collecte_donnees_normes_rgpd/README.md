# Portfolio-Data

Projet 4 : Collecter des données aux normes RGPD

Projet visant à collecter des données en se conformant aux normes RGPD. Dans un premier temps, un extraction de données était faite grâce au langage SQL. Puis, avec Power Query il fallait anonymiser ces données pour être en accord avec la réglementation.

Outils utilisés : SQL, Power Query 
Notion : RGPD
