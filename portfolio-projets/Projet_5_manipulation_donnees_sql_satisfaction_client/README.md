# Portfolio-Data
Projet 5 : Manipuler des données avec SQL pour évaluer la satisfaction client 

Ce projet avait pour but d'utiliser SQL pour comprendre la clientèle d'une magasin et améliorer l'expériene client. 
Nous avons calculés plusieurs indicateurs :
- NPS
- Nombre de retours clients
- Note moyenne accordée par les clients
- Pourcentage de recommandations clients

Outils : SQL
