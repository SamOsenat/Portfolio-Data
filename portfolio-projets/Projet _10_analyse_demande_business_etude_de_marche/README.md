# Portfolio-Data
Projet 10 : Analyse d'une demande business et étude de marché

Dans ce projet, nous avons du procéder à une étude de marché sur les jeux vidéos pour une entreprise souhaitant créer un jeu Triple A. Il fallait évaluer la pertinence de ce projet et effectuer des recommandations business grâce aux données fournies mais également grâce à d'autres données que nous avons pu récolter (données API par exemple).

Outils : Python, API
