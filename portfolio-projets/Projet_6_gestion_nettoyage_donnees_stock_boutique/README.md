# Portfolio-Data
Projet 6 : Optimiser le gestion des données stock d'une boutique 

Ici, on devait aider une entreprise à réconcilier les données issues de son ERP avec celles de son site Web.
Il fallait nettoyer les données afin de supprimer les doublons qui étaient présents.
Par la suite, une fois les données propres, on a pu analyser les données et visualiser le top 20 produits par exemple, ou encore calculé la rotation des stocks.
